Ruby Enumerables Cheat Sheet

Methods Code Description
#map map {|x|...} Returns an array with the results from block
#include? [1,2,3,4].include?(4) Returns true if any member of enum equals obj. Equality is tested using ==
#each_slice (1..9).each_slice(3) { |x| print x }

#=> [1, 2, 3][4, 5, 6][7, 8, 9]

 Breaks collection into a group of n sized elements. Useful to transform an array into an array of sub arrays for modifications.
#inject (1..5).inject { |sum, element| sum + element }

#=>15

Easily allows users to add, multiply, subtract through each element of a collection.
#sort sort [{|a, b| â€¦}] Returns array sorted by <=> operator or by block. Default block { |a, b| a <=> b }.
#minmax ["albatross", "dog", "horse"].minmax { |a, b| a.length <=> b.length}

#=> ["dog", "albatross"]

 Returns a two element array containing the objects in enum that correspond to the minimum and maximum values respectively from the given block.
