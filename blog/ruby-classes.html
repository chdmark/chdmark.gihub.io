Instances!!

Classes, instance variables, and instance methods. What are they? They can be confusing, but oh can they make your life a bit easier. In this blog post, I will show you how to create a class, instance methods, and instance variables. In addition, you will be able to observe how to call these methods outside of the class. Before we get into the code, let’s talk about what exactly is a class.

Classes usually consists of different method definitions within itself. The purpose of a class is to be instantiated. This means that there are objects that are instances of the class. The way to define a class is quite simple. We are going to define a simple die class:

class Die #Create a class by typing class

end

As you can see, you create a class by typing class and whatever name you want to call it. As I mentioned earlier, classes contain instance variables, method definitions within it, and instance methods. What exactly is an instance variable? An instance variable is much like a local variable; however, you can use that variable throughout your class and within different method definitions within your class. Instance variables usually start with an @ sign. In addition, you can initialize a class with an initialize method. In the code below, I will show you how to use the initialize method. When you initialize a method, the method will be used each time you create a class:

class Die #Create a class by typing class

def initialize(sides)

@sides = sides

end

end

We created the initialize method so that when you pass in a number to the class, that method will always run. Right now, our code would just return the value of the number we pass into it. As you can see, we also defined an instance variable. Let’s make this class a little more exciting by putting more methods within the class. You will be able to see how you can use the instance variable in other methods as well.

class Die #Create a class by typing class

def initialize(sides)

@sides = sides

end

def sides

@sides #We used the instance variable @sides from the first method

end

def roll

rand(1..@sides) #Generating a random number using rand

end

end

We were able to use the instance variables throughout different method definitions. It is important to define an instance variable when you want to use the same variable in other methods within the class. Remember that a local variable is limited to the scope of the method definition. I hope this gives you a better understanding of how classes and instance variables work! I’m still learning myself, so keep practicing!
